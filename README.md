# Round Table Sneaky Hands TrialRun

Friend-test static web prototype for a two-player top-down party game.

## Controls

- Player 1: `W`, `A`, `S`, `D`
- Player 2: arrow keys
- Touch/mobile: on-screen direction buttons
- Click/tap the table to send the nearest hand toward that point

Move slowly near the animals. Sudden movement scares them and makes them
scatter. Catching an animal scores a point.
