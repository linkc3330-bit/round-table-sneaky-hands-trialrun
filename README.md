# Round Table Sneaky Hands TrialRun

Friend-test static web prototype for a two-player top-down party game.

## Controls

- Right-click and hold near a hand lane to advance that hand toward the center.
- Release to pull the hand back.
- Touch/mobile: hold the P1/P2 lane buttons.

Move slowly near the animals. Holding too long turns the approach into a fast
lunge, scares the animals, and makes them scatter. Slow contact catches an
animal and scores a point.
